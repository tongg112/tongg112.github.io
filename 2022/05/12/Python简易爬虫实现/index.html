准备工作

使用 python3 环境，用 pip3 安装需要用到的 modules

爬取思路

  • 打开需要爬取的网站，一般来说，图片网站都会有：分类、图集、页码，在浏览器中找到这些元素
  • 第一步：获取全部图集信息
    • 图集名称、图集编号、图集地址、图集封面
  • 第二步：将图集信息储存在数据库中或Excel中
  • 第三步：从数据库中获取 url，对图集下的图片进行爬取下载，图集下载成功后标记数据库图集为已爬取状态
