## 教材详情

### 私（わたし）はアンナです。
- アンナ: はじめまして。私はアンナです。 // 	初次见面。我是安娜。
- さくら: はじめまして。さくらです。 // 初次见面。我是小樱。
- アンナ: よろしくお願いします。 // 请多关照。
- さくら: こちらこそ。 // 我也要请你多关照。

### 语法要点
A は B です。 // A 是 B。
> 「は」是表示主题的助词。  
「です」接在B后面，构成谓语。  
例如，「私はアンナです」（我是安娜）。  

### 单词

| はじめまして                                                 |
| ------------------------------------------------------------ |
| **初次见面**                                                 |
| 这是对初次见面的人说的寒暄语。如果有人对你说「はじめまして」，你可以回答说「はじめまして」。 |

| 私                                                           |
| ------------------------------------------------------------ |
| **我**                                                       |
| 「よろしくお願いします」是自我介绍时经常说的一句话，意思是「希望我们今后保持良好关系」。除此以外，在有事拜托对方的时候，也会在最后附加上这句话。 |

| よろしくお願いします |
| -------------------- |
| **请多关照**         |

| こちらこそ                |
| ------------------------- |
| **我也···、是我该这么说** |
