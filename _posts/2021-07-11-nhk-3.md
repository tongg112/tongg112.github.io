# NHK《简明日语》第03课
## 课程简介
今天，安娜要跟小樱去学校看看。
## 教材详情

### トイレはどこですか。
- さくら: ここは教室です。	 // 这里是教室。
- アンナ: わあ、広い。	 // 哇，好大。
- さくら: あそこは図書館。	 // 那里是图书馆。
- アンナ: トイレはどこですか。	 // 洗手间在哪儿？
- さくら: すぐそこです。	 // 就在那儿。

### 语法要点
____はどこですか。  // （···在哪儿？）
> 「どこですか」是询问场所的说法。  
例如，「トイレはどこですか」，意思就是“洗手间在哪儿？”
