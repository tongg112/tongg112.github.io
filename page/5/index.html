发表于2021-07-192025-08-04 excel
Excel使用公式某一列乘以固定单元格在制作 excel 清单时使用公式：[H4 单元格]：=F4*G4然后拉动 H4 单元格自动填充 H5 到 H10，则：[H5 单元格]：=F5*G5[H6 单元格]：=F6*G6… …[H10 单元格]：=F10*G10但是我需要让 H 列等于 F 列乘以 G4 单元格，而不是乘以 G 列。 上网查资料找到解决办法：[H4 单元格]：=F4*G$4然后再拉动 H4 单元格自动填充，就可以了。 在要固定的单元格数字前加上$符号。 参考资料 excel乘固定单元格 header.intro-header { background-blend-mode: multiply; }
